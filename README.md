# 基础图形绘制软件使用手册

本手册旨在指导用户使用基于 Python 与 Tkinter 开发的基础图形绘制软件。该系统支持多种几何图形的创建、编辑、撤销/重做、组合、精准尺寸调整以及文件存取等操作。

## 一、 系统环境要求与运行步骤

### 1.1 环境要求
*   **操作系统**：Windows、macOS 或 Linux。
*   **运行环境**：Python 3.8 或更高版本。
*   **依赖库**：系统仅使用 Python 标配的标准库（包含 `tkinter`、`json`、`math`、`copy` 等），无需额外安装第三方依赖。

### 1.2 启动步骤
1. 将重构后的代码按前述目录结构放置在本地项目文件夹中（例如 `drawing_software/`）。
2. 打开终端或命令提示符（Terminal / CMD），进入该文件夹根目录。
3. 执行以下命令启动软件：
   ```bash
   python main.py
   ```

## 二、 界面布局介绍

主界面由四个主要区域构成：
1.  **顶部工具栏 (Top Bar)**：
    *   `Undo` / `Redo`：撤销与重做按钮。
    *   `置顶` / `置底`：用于调整图形的渲染层级（叠放次序）。
    *   `填充色` / `透明背景` / `描边/文字色`：用于配置当前画笔或选中图形的颜色。
    *   `文本内容` / `设置文本`：配置和修改独立文本框的文字。
    *   `组合` / `打散` / `删除`：管理多个图形的组合与生命周期。
    *   `暗黑模式`/`亮白模式`：切换界面主题样式。
    *   `文本字号`：调整文本框的字体大小。
2.  **左侧工具栏 (Sidebar)**：包含“选择工具”以及“矩形”、“圆形”、“椭圆”、“三角形”、“连接线”、“文本框”六种基础图形的绘制工具。
3.  **中央画布 (Canvas)**：执行图形绘制、框选、拖拽和缩放的交互区域。
4.  **底部状态条 (Status Bar)**：实时显示操作提示、当前工具状态及选区信息。

## 三、 功能操作指南

### 3.1 图形绘制与文本添加
1.  **绘制图形**：在左侧工具栏点击选中对应的图形工具（如 `方框矩形`），在中央画布上按下鼠标左键并拖拽，拉出期望的大小后松开左键，即可完成绘制。
2.  **添加独立文本框**：
    *   点击左侧的 `文本框` 工具，在画布上拖拽绘制。
    *   绘制完成后，选中该文本框，在顶部工具栏的“文本内容”输入框中键入文字，点击 `设置文本` 按钮。
    *   亦可通过顶部工具栏的“文本字号”微调框来改变该文本框的字体大小。

### 3.2 选中、多选与删除
1.  **单选图形**：切换到左侧的 `选择工具`，直接用鼠标左键点击画布上的任意图形。
2.  **多选图形**：
    *   **框选**：在画布空白区域按下鼠标左键并拖动，拉出的蓝色虚线框接触到的图形会被一并选中。
    *   **加选/减选**：按住键盘 `Shift` 键的同时点击图形。
3.  **删除图形**：选中图形后，点击顶部工具栏的 `删除` 按钮，或直接按键盘的 `Delete` 键。

### 3.3 图形变换与尺寸调整
1.  **移动图形**：在使用选择工具选中一个或多个图形后，按住图形并在画布上拖拽鼠标，即可进行平面移动。
2.  **拖拽缩放**：
    *   点击选中目标图形，图形右下角会出现一个蓝色控制点。
    *   按住该控制点拖动鼠标，可实时调整图形的尺寸。
3.  **精准数值微调**：
    *   使用选择工具，对单个图形（如矩形、圆形等几何图形）点击鼠标右键（macOS 用户可使用双指点击或 `Button-2`）。
    *   弹出的对话框中会显示该图形的当前宽度和高度，输入期望的新数值（正整数），点击“确认”即可精准应用。
    *   *注：组合图形和文本框暂不支持右键数值调整。*

### 3.4 拷贝复制与粘贴
1.  选中一个或多个图形，使用键盘快捷键 `Ctrl + C`（或在“编辑”菜单中点击“复制”），系统会将图形对象克隆保存至内存剪贴板。
2.  按下 `Ctrl + V`（或在“编辑”菜单中点击“粘贴”），克隆的图形会产生偏移动作并加入画布。用户可连续粘贴。

### 3.5 组合与解组
1.  使用框选或 `Shift` 键选中两个或更多图形。
2.  点击顶部工具栏的 `组合` 按钮。组合后，所有子图形将作为一个整体，支持整体移动、缩放和克隆。
3.  若要恢复为独立图形，选中组合体后点击 `打散` 按钮。

### 3.6 多步撤销与重做
*   系统自动记录每一次绘制、删除、移动、缩放、层级微调及组合操作。
*   点击顶部工具栏的 `Undo` 按钮（或快捷键 `Ctrl + Z`）可撤销上一步操作。
*   点击顶部工具栏的 `Redo` 按钮（或快捷键 `Ctrl + Y`）可恢复已撤销的操作。

### 3.7 文件导入与导出
*   **保存**：点击菜单栏的 `文件 -> 导出保存 (Save)`，可将当前画布的状态及所有图形层级以序列化的 JSON 文件格式保存至硬盘。
*   **加载**：点击菜单栏的 `文件 -> 导入图形 (Load)`，选择已保存的 JSON 矢量文件，可恢复先前的绘制状态。
